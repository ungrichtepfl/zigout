# cout

The C Port of the __zigout__ program.

A makefile for [GNU Make](https://www.gnu.org/software/make/) is include for easy builds:

To build:
```
RELEASE=1 make
```

To run: 
```
RELEASE=1 make run
```
 
**Dependencies:**

- SDL2 and SDL2-TTF (Ubuntu: `sudo apt install libsdl2-dev libsdl2-ttf-dev`)
